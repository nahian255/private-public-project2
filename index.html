    • what is difference between var, let and const?

      var variables can be updated and re-declared within its scope;
      let variables can be updated but not re-declared;
      const variables can neither be updated nor re-declared.

    • what is difference between arrow function
      and normal function?

      Unlike regular functions, arrow functions do not have their own this.
      The value of this inside an arrow function remains the same
      throughout the lifecycle of the function
      and is always bound to the value of this
      in the closest non-arrow parent function.
